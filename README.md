## This project is a JavaScript-based website utilizing the page.js and lit HTML libraries. It serves as a platform for electrical services.

## Features:
- Service Menu: detailed listing of available electrical services;
- Past Projects: showcase of previous electrical projects completed;
- Pricing: clear pricing information for services offered;
- Contact Form: form for users to get in touch with inquiries;

## Technologies Used:
- JavaScript
  - page.js
  - lit-html
- HTML
- CSS
  
## Usage Instructions:
In the project directory, you can run:
- npm start
  
Runs the app in the development mode.
- Open http://localhost:3000 to view it in your browser.
